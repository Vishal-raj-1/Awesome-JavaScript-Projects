JS Slider Get in touch
JS Slider is a javascript project to add animation in your navigation bars.
One can use it as sidebar menus or option drawers.

						API used:
						

						


							Open and close slider pushbar with api
							
						
					

						Events
						

						


							
'pushbar_opening': when a slider pushbar is opening, it will emit 'pushbar_opening' event with an object { element, id }, where element will be the current pushbar element and the id is the current pushbar id.

							
'pushbar_closing': when a slider pushbar is closing, it will emit 'pushbar_closing' event with an object { element, id }, Where element will be the current pushbar element and the id is the current pushbar id.
